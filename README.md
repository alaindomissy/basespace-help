_config.yml
-------------
`_config.yml` stores configuration data. 


| Variable					| Description														|
| --------------------- | -----------------------------------------------|
| gitrepo						| URL to the project's GitHub repo. *Used to build the edit on GitHub link.*				|
| gitbranch				| Git branch name. *Used to build the edit on GitHub link.*											|
